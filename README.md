# Mesto-ad

Проект `mesto-ad` представляет собой адаптивную фронтенд-страницу с функциональностью работы с карточками мест и профилем пользователя. Он разработан на основе современного стека веб-технологий и показывает практическое использование модульной разметки, CSS и JavaScript.

## Технологии
- HTML
- CSS
- JavaScript
- Vite
- Fetch API

## Как запустить
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите локальный сервер разработки:
   ```bash
   npm run dev
   ```
3. Откройте приложение в браузере по адресу, указанному в терминале.

## Как собрать
```bash
npm run build
```
